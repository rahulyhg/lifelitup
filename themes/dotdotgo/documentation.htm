F.A.Q

- What is the image size for the home page?

The image size is 928 by 250.

- How many items can I enter in the Scrollable Gallery and Nivo Slider?

You can enter as many items as you like. If you are entering more than four images in the Scrollable Gallery then please edit the margin for following CSS declaration so that the navigation is centered. The specific part is in bold below:

.navi { background: #16bef9 url(images/gradient.png) repeat-x left bottom;
border: 1px solid #0795D1; float: left; height: 12px; margin: -22px 0 0 379px;
padding: 15px 0; position: relative; z-index: 999;
-moz-box-shadow: 0 0 5px 0 #000; -webkit-box-shadow: 0 0 5px #000;
-moz-border-radius: 3px; -webkit-border-radius: 3px; }

- How do I use the Contact form?

Edit your contact page information by opening up sendEmail.php and change the email address and site owners name.

Example:

$site_owners_email = 'you@site.com'; // Replace this with your own email address
$site_owners_name = 'Your Form Name'; // replace with your name

- How do I enter my newsletter signup form?

Simply enter the code provided by your newsletter service provider. The CSS should style it for you, though you may need some tweaking.
